1. Classes

- Can also be used by a JavaScript to access and manipulate elements with the specific class name.
- The class attribute can be used on any HTML element.
- To create a class; write a period ( . ) character, followed by a class name.
- HTML elements can belong to more than one class.
- To define multiple classes, separate the class names with a space, e.g. class="city main".
